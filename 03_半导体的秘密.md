# 第三章：半导体的秘密

## 本章属于「器件」维度——底层

前面两章讲完了信号怎么描述、怎么用无源网络塑造。但从这一章开始，方向转了——从"信号"转向"器件"。因为要突破无源世界的天花板（信号只能变小不能变大），你需要一种"能主动控制电流"的东西。

本章回答第一个器件问题：**凭什么存在"有时导电、有时不导电"的材料？** 这是晶体管诞生的物理前提。

---

## 1. 能带——为什么有的东西导电，有的不导电

原子里的电子不是随便飞的。每个电子只能待在某些特定的"能量楼层"上——这叫能级。当大量原子凝聚成固体时，相邻原子的能级互相影响、展宽、变成近似连续的"能带"。

导电的关键看两个带：

- **价带**：正常情况下电子填满的带——电子被原子核束缚着参与化学键
- **导带**：价带上面更高的带——电子到了导带就是自由电子，能满材料跑，参与导电
- **禁带（带隙 / bandgap）**：价带和导带之间的距离——电子从价带跳到导带需要跨过的"能量鸿沟"

按带隙宽度分三类材料：
- **导体（金属）**：价带和导带重叠，电子随时自由——天生导电
- **绝缘体**：带隙 > 5eV，价带电子要吸收极大能量才能跳到导带——正常条件下几乎不导电
- **半导体**：带隙适中（硅 1.12eV）——正常温度下少量电子有足够热能跳到导带，导电性介于导体和绝缘体之间；更重要的是，**它的导电性可以被外部手段调控**，这是导体和绝缘体都做不到的

**为什么是硅？** 硅的带隙 1.12eV——不算太小（不至于漏电太大）、不算太大（不至于太难导电）。储量丰富（沙子），工艺成熟，而且自然氧化层 SiO₂ 是优良绝缘体（在芯片上做绝缘隔离很容易）。高频/高功率/光电器件用 GaAs、GaN、SiC——它们的带隙和电子特性不同，各有各的适用场景。

序言里讲过一个信号层面的理由：**带隙直接决定信号在关断状态下的泄漏大小。** 带隙越宽，电子从价带热激发到导带的概率越低，晶体管关断时的漏电流越小。硅的带隙比锗宽（1.12eV vs 0.67eV），所以硅晶体管在关断时信号泄漏低得多——后来集成电路选硅而不是锗，这是信号质量要求做出的物理选择。

### 晶格——电子的"路"长什么样

上面讲的是能带——电子被允许待在哪里。但还有一个前提问题：电子走的路本身长什么样？

半导体器件必须用**单晶硅**——整块材料里所有硅原子排成完全规整的周期性阵列，每个硅原子和周围四个邻居组成一模一样的正四面体（金刚石结构）。为什么必须是单晶？因为只有周期完美的晶格，才能让几十亿个晶体管的导电特性一模一样。多晶硅（很多小晶粒拼起来）的晶粒边界是电子的"陷阱站"——电子走到晶界上被捉住、散射、复合，产生噪声和不可控的电阻。

即使最好的单晶硅也做不到完美。三种缺陷：

- **点缺陷**：格点上少一个原子（空位），或多一个不该在那里的原子。在带隙中引入局域的陷阱能级——第 8 节 SRH 复合的中心站
- **位错**：一整排原子偏离理想位置，沿线一串深能级复合中心
- **界面态**：硅和二氧化硅界面上的硅原子不再有四个完整邻居——剩下的悬挂键是天然的深能级陷阱，1/f 噪声的物理源头（第 5 章）

从信号视角一句话：**电子在完美晶格里像走高速路——路平、没坑。缺陷就是路上的坑——撞上去被弹回来（散射），或者掉进去出不来再随机爬出来（1/f 噪声）。**

晶体是各向异性的——不同方向原子排列疏密不同。硅晶圆几乎全切 (100) 面：这个面暴露的悬挂键最少 → 和 SiO₂ 的界面陷阱最少 → 1/f 噪声最低 → 阈值电压最稳。晶向不是"学术细节"——它直接写进了噪声预算。

## 2. 掺杂——把半导体变成"可调控"的导体

纯硅（本征半导体）导电性很差——室温下热激发到导带的电子太少了，载流子浓度极低。

但硅是四价元素——每个硅原子伸出四个电子跟四个邻居组成共价键。如果你往硅晶体里塞一点五价元素（磷、砷），多出来的那一个电子没有化学键可参与，孤零零地飘在晶格中，极容易被激发到导带——这就是**N 型半导体**，靠电子导电。那个"多出来的电子"是多数载流子。

如果你塞三价元素（硼），硅原子上缺了一个电子——那个空缺就像一个"空座位"，邻居的电子可以跳过来填上，但原来的位置又空了，等效于空位在移动。这就是**P 型半导体**，靠空穴导电。空穴等效于带正电的粒子——它是多数载流子。

掺杂浓度几个数量级地改变了半导体的导电性——而且你可以精确控制掺杂的浓度和位置。这就是半导体成为"可编程材料"的原因。

### 施主能级与受主能级——为什么浅能级杂质好用

磷那个"多出来的电子"不是本来就自由——它待在一个特定的**施主能级**上，离导带底只有 0.045eV。室温热能 kT ≈ 0.026eV——两个数在一个量级，所以施主上的电子极容易被"热摇"进导带。硼引入的**受主能级**同样道理，离价带顶也只有 0.045eV，价带电子极容易跳上去，在价带留下自由空穴。

磷和硼都是**浅能级杂质**——引入的能级离能带边缘极近，室温下几乎 100% 电离。掺杂浓度 = 载流子浓度。

反过来，金（Au）、铂（Pt）在硅带隙中引入的能级靠近中央——**深能级杂质**。室温下大部分没电离，不贡献载流子。但它们极擅长做另一件事：做电子和空穴配对湮灭的"中间站"——第 8 节的 SRH 复合。深能级杂质是少数载流子寿命的杀手。

从信号视角看：浅能级让你精确控制"这管子通多少电流"。深能级是所有模拟设计师生不如死地恨的东西——它在你没法预测的地方加噪声、加漏电。

## 3. 漂移与扩散——两种驱动方式

载流子在半导体里怎么运动？两种基本方式：

- **漂移**：存在电场时，电子被电场拉着往正电位走，空穴被拉着往负电位走。电流密度 = σE。这是外加电压时的主要运输方式。
- **扩散**：没有电场也行——载流子从浓度高的地方自发往浓度低的地方散开，就像墨水滴进水里。扩散电流正比于浓度梯度。

这两种运动方式——漂移是被动（外场拉着走），扩散是主动（浓度差推着走）——共同构成了半导体器件内部所有电流的物理来源。PN 结的工作原理就是漂移和扩散的博弈。

### 迁移率——载流子到底能跑多快

漂移速度 v = μE。μ 就是迁移率——单位电场下载流子的平均漂移速度，整个半导体设计里最核心的数字之一。迁移率大 → 同样电场下电流大 → 同样偏置下跨导 gm 大。

但迁移率不是常数。载流子在晶格里被两种障碍不断散射：

- **晶格散射（声子散射）**：温度越高 → 晶格振动越剧烈 → 迁移率下降。μ ∝ T^(-3/2)——温度从 25°C 升到 85°C，迁移率掉约 30%，gm 跟着掉
- **电离杂质散射**：掺杂的磷原子电离后带一个正电——这个固定的正电中心对经过的电子施加库仑引力。掺杂越重 → 散射中心越多 → 迁移率越差

**总迁移率由其中更差的那个限制。** 轻掺杂区晶格散射主导，重掺杂区电离杂质散射主导。

扩散系数 D 和迁移率 μ 之间由爱因斯坦关系锁定：D/μ = kT/q。**迁移率高的材料扩散也快——漂移和扩散本质上是同一件事（载流子在晶格中的运动能力）在两种驱动力下的不同面孔。**

## 4. 费米能级——电子的"水位线"

前面三节讲了能带、掺杂、漂移扩散。但有一个问题还没回答：**掺杂之后，到底有多少电子在导电？** 回答这个问题需要引入一个统计量——费米能级。

费米能级经常被误解成"禁带中间那条线"。它不是。它是一个概率分界——**电子占据一个能量为 E 的量子态的概率 = 50% 的那个能量。**

物理直觉：一个水箱的水面线——水面以上大概率是空的，水面以下大概率是满的。费米能级就是电子的那条"统计水面线"。

### 费米能级的位置由掺杂决定

- **本征半导体**：电子完全来自价带到导带的热激发——每多一个电子就多一个空穴，对称 → 费米能级在禁带正中央
- **N 型**：施主塞了大量额外电子 → "电子水位"被推高 → 费米能级从中央移向导带底。掺杂越重，离导带越近
- **P 型**：受主从价带抽走电子（=注入空穴）→ "电子水位"被拉低 → 费米能级移向价带顶

### 费米能级统一了解释

三条线索在这拧成了一根绳：

1. **掺杂类型**——N 还是 P，不看掺了什么元素，看费米能级在禁带中央的哪一侧
2. **载流子浓度**——费米能级位置 + 费米分布 → 积分出导带电子和价带空穴各有多少
3. **PN 结内建电势**——N 型费米能级高，P 型费米能级低。贴在一起时，电子从高水位流向低水位，产生的内建电场让能带弯曲——平衡时两端费米能级拉平。Vbi（硅 ~0.7V）= 两边初始费米能级差。第 4 章 PN 结的那个 0.7V，物理来源就在这一节

### 有效质量——为什么不同材料里电子"轻""重"不同

电子在真空里：F = m₀a。电子在晶格里：晶格周期势场像一片起伏的地形，电子穿行时不断被格点正离子拉拽，它的"有效惯性"不再是 m₀。

能带曲率大 → 有效质量小 → 电子容易被加速 → 迁移率高。GaAs 电子有效质量 0.067 m₀——电子极轻，高频性能最好。硅 0.26 m₀——中等。有效质量小是 GaAs 和 InP 统治射频和光电器件的物理根源。

材料选择不是工程师的偏好——是晶格结构通过有效质量把速度天花板写死了。

## 5. 平衡载流子——n×p = ni²

### 多少个电子在导电

导带里有多少个电子 = "每个能量上的座位数（状态密度）" × "坐人的概率（费米分布）"，从导带底往上积分。费米分布在导带底以上快速衰减——电子全泡在导带最底部，越往上越少。

积分结果凝成一个极简洁的关系——**质量作用定律**：

**n × p = ni²**（给定温度下常数）

硅 300K：ni ≈ 1.5×10¹⁰ cm⁻³。

这个公式的意思是：**热平衡下，电子浓度和空穴浓度互相制约——你没法同时增加两者。** N 型掺杂把 n 从 ~10¹⁰ 推到 ~10¹⁵——电子翻了十万倍；p 同时被压到 ni²/n ≈ 2.25×10⁵ cm⁻³——只有原来的百万亿分之一。多数载流子和少数载流子差了十个数量级。

少数载流子少到几乎可以忽略——**但它们就是 PN 结反向偏置时漏电流的物理承担者。** 为什么锗的漏电比硅大？锗的 ni（300K）约 2.4×10¹³——ni² 比硅大六个数量级 → 少数载流子浓度高六个数量级 → 反偏漏电流大六个数量级。

### 温度——信噪比的隐形杀手

ni 随温度指数增长：ni ∝ T^(3/2) × e^(-Eg/2kT)。室温附近，温度每升高约 10°C，ni 大约翻一倍。

芯片 70°C（手机 SoC 跑游戏时的正常结温）→ ni² 比室温高几十倍 → PN 结反向漏电流飙几十倍。125°C（汽车/工业规格高温极限）→ ni² 比室温高约三个数量级。从信号视角看：温度升高不改变信号大小，但抬高本征载流子浓度 → 抬高反向漏电流 → 抬高噪声地板。

## 6. 强电场——速度饱和

低电场下 v = μE——欧姆定律区，速度随电场线性增长。但电场超过约 10⁴ V/cm 后，载流子从电场获得的动能大到每次和晶格碰撞都激发高能光学声子——能量几乎全部丢掉，重新加速再撞——等效速度不再增长，饱和了。

硅电子饱和速度 ≈ 10⁷ cm/s。100nm 沟道 + 1V 漏电压 → 平均电场 ~10⁵ V/cm → 早已进入速度饱和。短沟道 MOSFET 的电流在正常偏置下就工作在速度饱和——不再随 Vds 平方增长，而近似线性依赖于 Vgs - Vth。

从信号视角：**速度饱和是摩尔定律撞上的最底层物理墙之一。** 沟道可以缩到 3nm，但载流子本身的饱和速度不会变——10⁷ cm/s 是电子在硅里的速度上限。信号在单管层面的延迟下限由这个数划出。

## 7. 非平衡载流子——从静到动

前面第 4~5 节讨论的全是**热平衡**——没有外部能量注入，n×p = ni² 严格成立。但你一旦给 PN 结加电压、用光照射半导体、或者管子开始工作——热平衡立刻被打破。非平衡态才是"器件被使用"时的常态。

### 注入与复合

PN 结正偏时，N 区电子被注入 P 区——P 区（原本空穴是多数、电子是少数）的少数载流子电子浓度瞬间飙了几个数量级。这叫**少数载流子注入**。外加激励撤走后，多出来的电子-空穴对通过三种方式配对消失：

- **辐射复合**：电子跳回价带，放出一个光子。直接带隙材料（GaAs）主导——LED 和激光器的物理基础。硅是间接带隙，辐射复合效率极低，几乎不发光
- **俄歇复合**：电子-空穴复合的能量传给第三个载流子——高掺杂/高注入时主导
- **SRH 复合（Shockley-Read-Hall）**：通过带隙中深能级陷阱作为"中间站"——电子先跌进陷阱，空穴再跌进去配对，能量分两次以热形式释放。**硅作为间接带隙材料，SRH 复合是非平衡载流子消失的主要路径**

### 复合寿命 τ 与扩散长度 L

非平衡少数载流子的平均存活时间 = τ（复合寿命），在复合前平均扩散的距离 = L = √(Dτ)。

τ 反比于深能级陷阱浓度。缺陷多的硅 → τ 短 → 注入的少子走不远就死了 → BJT 基区输运效率掉（第 4 章）、光电探测器量子效率掉（第 9 章）。τ 长 → 少子走得远 → 增益高、效率高——但开关速度慢（关断时少子清空需要更长时间）。

硅（间接带隙）的 τ 可以从纳秒到毫秒，少子可以走几百 μm。GaAs（直接带隙）的 τ 只有纳秒到皮秒，少子几乎走不动——这就是为什么 BJT 在硅上能做好、在 GaAs 上难做：基区宽度必须小于少子扩散长度。

### 准费米能级——信号的"物理签名"

热平衡下电子和空穴共用一个费米能级 E_F。非平衡下——电子浓度被推高、空穴浓度也被推高——再共用一个 E_F 物理上不可能（同一个数不能同时靠近导带又靠近价带）。解决：给电子和空穴各分配一个**准费米能级**——E_Fn 和 E_Fp。

E_Fn 和 E_Fp 的分裂量 = 非平衡注入的强度。给 PN 结加正向电压 V → 结两端各自的 E_Fn 和 E_Fp 被拉开 qV 的能量差 → 第 4 章 PN 结 I-V 方程 I = I₀(e^(qV/kT) - 1) 里，指数上的 V，物理意义就是 E_Fn - E_Fp。

从信号视角：**准费米能级的分裂量 = 你加在 PN 结上的电压信号。** 第 5 章你用差分对放大的 Vgs，物理上就是 E_Fn - E_Fp 的差值。信号通过半导体——不是抽象的"电压变电流"，而是费米能级在空间上的分裂和拉平。

### 连续性方程——半导体版的 KCL

载流子不会凭空出现或消失。在任何一个微小体积内：**载流子浓度净变化率 = 流入流出净差 + 产生率 - 复合率。**

这是半导体器件分析的基本运动方程。第 4 章 PN 结的 Shockley 方程、BJT 的 Gummel-Poon 模型、MOSFET 沟道电流——全部是从连续性方程 + 漂移扩散电流方程 + 泊松方程三联立解出来的。

不必记方程形式，只记一件事：**连续性方程把漂移、扩散、产生、复合写在了同一个等式里。** PN 结耗尽区里扩散可忽略，产生主导 → 反向饱和电流由耗尽区 SRH 产生率决定。PN 结正偏准中性区漂移可忽略，扩散主导 → 正向电流由注入少子的扩散输运决定。第 4 章进 PN 结，这些术语已经在手边了。
